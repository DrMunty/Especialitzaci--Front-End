» Com s'integra el control de versions (Git) dins de l'entorn de VS Code?

VS Code detecta automàticament quan obres una carpeta que és un repositori de git i activa tots els controls de versions. El programa ofereix les següents opcions:

- Inicialitzar un nou repositori: Crear un nou repositori de Git a la teva carpeta.
- Clonar un repositori: Clonar un repositori existent de GitHub o un altre host de Git.
- Obrir un repositori remot: Treballa amb un repositori sense clonar-lo a la teva màquina local gràcies a l'extensió GitHub Repositories.

Gràcies als elements integrats de la UI de Visual Code pots realitzar operacions Git sense necessitat de fer servir la terminal:

- Centre de control de versions: Des d'aquí pots realitzar operacions de Git com staging, committing i controlar tots els canvis.

- Gràfic de control de versions: Representació gràfica del teu historial de commits i relacions.

- Diff editor: Comparacions en temps real per poder realitzar els canvis necessàris.
